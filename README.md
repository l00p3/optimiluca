# Optimiluca
Optimiluca: Pose Graph Optimization Library by Luca

## Requirements
 - gcc >= 13.1.0
 - cmake >= 3.22.1
 - eigen >= 3.4.0
 - libtbb >= 2021.5.0-7

## Usage
```bash
make run
```

## Statistics
 - Current Average Iteration Execution Time (1000 states, 1000 closures): 0.025 sec
